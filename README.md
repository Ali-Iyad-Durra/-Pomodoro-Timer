# 🍅 Pomodoro Timer
**Developed by Ali Durra**

A sleek, modern Pomodoro Focus Timer built with **WPF (.NET)**. Designed for developers, gamers, and creators who need a distraction-free environment to boost their productivity with style.

---

## 🚀 Key Features

* **Fluent Design UI:** Modern interface with rounded corners, smooth drop shadows, and a clean layout optimized for Windows 11.
* **Dual Theme System:** Seamlessly toggle between **Dark Mode** and **Light Mode** to suit your environment.
* **Professional Timing Logic:**
    * **Focus Session:** 25 minutes of deep work.
    * **Break Session:** 5 minutes of rest.
    * Automatic transitions between states.
* **Focus Mode (Full Screen):** Maximize the app to fill the screen and remove all border distractions.
* **Audio Alerts:** Clear audio notifications when a session ends so you never miss a transition.
* **Lightweight & Fast:** High performance with minimal RAM usage, ensuring it doesn't interfere with your gaming or dev tools.

---

## 🛠️ Technical Stack

* **Language:** C#
* **Framework:** WPF (Windows Presentation Foundation)
* **Styling:** XAML (Custom Control Templates)
* **Deployment:** Inno Setup (Professional Installer)

---

## 📥 Installation

1.  Navigate to the **[Releases](https://github.com/YourUsername/PomodoroTimer/releases)** page.
2.  Download the latest `PomodoroTimer_Setup.exe`.
3.  Run the installer and follow the simple on-screen instructions.
4.  Launch the app and start focusing!
